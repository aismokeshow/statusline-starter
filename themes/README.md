# Theme Presets

Drop-in color palettes for the SMOKE statusline. Copy any theme's variables into `statusline-smoke.py`, replacing the `# ━━ SMOKE Palette` section.

| Theme | File | Vibe |
|---|---|---|
| Nord | `nord.py` | Arctic blues, muted and calm |
| Catppuccin Mocha | `catppuccin-mocha.py` | Soothing pastels on dark base |
| Dracula | `dracula.py` | High-contrast vivid purples |

Or use `/customize` in Claude Code to generate any palette on the fly.
